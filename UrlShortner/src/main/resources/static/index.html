Spring Boot + PostgreSQL + Redis

Shorten URLs instantly

Paste a long HTTP or HTTPS URL to generate a compact short link. Redirects are served through Redis-first resolution for low latency.

Only absolute HTTP/HTTPS URLs are accepted.

How it works

  • PostgreSQL creates a numeric ID
  • The ID is encoded to Base62
  • Redis stores the shortCode → originalUrl mapping

API endpoints

  • POST /shorten
  • GET /{shortCode}
  • GET /actuator/health
